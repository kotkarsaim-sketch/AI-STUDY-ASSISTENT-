## ✨ Features

| Feature | Description |
|---|---|
| 📄 **Smart Upload** | Drag-and-drop PDFs or paste your notes directly — text is extracted instantly on the client side |
| 📝 **AI Summaries** | Automatically generates concise, extractive summaries from your uploaded content |
| ❓ **Quiz Generation** | Creates multiple-choice quizzes from your material with instant scoring and feedback |
| 🃏 **Flashcards** | Produces interactive flip-cards for active recall — the gold standard of studying |
| 🌙 **Dark Mode Design** | Premium dark-mode-first UI with glassmorphism, gradients, and micro-animations |
| 📱 **Fully Responsive** | Looks great on desktop, tablet, and mobile devices |
| 🔒 **Privacy First** | All processing happens in your browser — your notes never leave your device |

## 🚀 Getting Started

### Prerequisites

- **Node.js** ≥ 18.x — [Download here](https://nodejs.org/)
- **npm** ≥ 9.x (bundled with Node.js)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/ai-study-assistant.git
cd ai-study-assistant

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the app.

### Production Build

```bash
npm run build
npm start
```

## 🎯 How It Works

```
┌──────────────┐     ┌──────────────────┐     ┌────────────────────┐
│  Upload PDF  │ ──> │  Extract Text    │ ──> │  Generate Results  │
│  or Notes    │     │  (Client-side)   │     │  Summary / Quiz /  │
│              │     │                  │     │  Flashcards        │
└──────────────┘     └──────────────────┘     └────────────────────┘
```

1. **Upload** — Drop a PDF file or paste your notes into the text area.
2. **Extract** — Text is extracted from the PDF entirely in the browser using `pdf.js`.
3. **Generate** — The app analyzes the text and produces:
   - A **summary** highlighting the most important sentences
   - A **quiz** with multiple-choice questions based on the content
   - **Flashcards** pairing key terms with their definitions
